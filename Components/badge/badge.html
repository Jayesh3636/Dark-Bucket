Badge

Badge is used where we want to show some notification/message count or show on an icon indicating there is some notification.

Badge can be of 3 different types

1) Badge with Avatar
2) Badge with Icon
3) Badge with Typography

Below examples illustrate the 4 different types of badges. To use any of these badges, simply add class badge followed by its type. Example → class="badge-online"

Badge with Avatar

Example → class="avatar" & span class="badge-away"

Badge with Icons

Example → class="badge-icons" & span class="notification"

Badge with typography

There are two sorts of text badges that may be used in combination with text. Badge with square and rounded corners. The badge text font size is measured in em. It will adjust according to the font-size of the parent element.
