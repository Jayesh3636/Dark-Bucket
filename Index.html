What is
Dark Bucket is a ready-to-use CSS library that comes pre-loaded with all of the necessary components for your next project. Built scalable web apps that provide you with easy access and high functionality.

Installation
Copy/Paste

Copy and paste the code in the head tag of your html document. So that you can style your components by just adding class names to your html elements.
