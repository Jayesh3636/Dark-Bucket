Typography

Typography means that you can alter the size and alignment of the fonts on your websites.

In typography there are different types like Headings, Small Texts, Alignment Text

The below examples illustrate the different types of typography. To add any one of these, simply add the class names.

Example of Heading text

You may use h1, h2, h3, h4, h5, and h6 components for headings.
The font-styling is the same for the class names h1, h2, h3, h4, h5, and h6. To style the text, use one of these classes.

Example of Paragraph text

You may use p tag components for paragraph
The same font-styling is used for the class name p, as well as the body-cp-rg class. To style the text, use one of these classes.Aside from that, if you want to change the size of the text, you may use the body-cp-lg or body-cp-sm classes.

P-Paragraph-large-24px/2.4rem

P-Paragraph-medium-16px/1.6rem

P-Paragraph-regular-14px/1.4rem

P-Paragraph-small-12px/1.2rem

Example of Aligned Paragraph text

To position the text, apply the text-center, text-left, and text-right classes to the wrapper element.
